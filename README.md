## Building

    mkdir build
    cd build
    cmake ..
    make

If libevent is installed in an obscure location (e.g. on Windows), you may
need to amend the `CMAKE_PREFIX_PATH` variable to help it find the headers and
libraries, e.g.

    cmake -DCMAKE_PREFIX_PATH=%HOME%\libevent
